# Code of Conduct

## The short version

Be a decent human. Reinicorn is a small project run by people; treat everyone
in issues, pull requests, and discussion with basic respect.

## Not tolerated

- Harassment, personal attacks, or discriminatory language.
- Spam, including automated bot submissions with no human intent behind them
  (see [CONTRIBUTING.md](CONTRIBUTING.md)).
- Deliberately disruptive or bad-faith behavior.

## Enforcement

The maintainer may edit, remove, or reject any contribution, and may block any
account, for conduct that violates this document. To report concerns privately,
open a GitHub private advisory
([Security → Report a vulnerability](../../security/advisories/new)) — it's a
confidential channel to the maintainer even for non-security matters.

That's it. If you're unsure whether something is okay, it probably is — just use
good judgment.
