# Security Policy

## Reporting a vulnerability

Please report security issues **privately** — do not open a public issue.

Open a private advisory via GitHub:
[Security → Report a vulnerability](../../security/advisories/new).

Include what you found, how to reproduce it, and the impact you expect. We'll
acknowledge your report and work with you on a fix and disclosure timeline.

## Supported versions

Reinicorn is pre-1.0. Only the latest release on `main` is supported; fixes are
not backported.
